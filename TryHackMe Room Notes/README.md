# Read Me
This folder contains the compilation of my TryHackMe room notes. 

# Structure
Every file is done in markdown though the structure may vary as I experiment with different layouts.

As of 20/03/2026 the current format is to have a summary and a notes section though not all summary sections have been filled for the relevant room.

# Progression
As I progress through the rooms I may elect to create more folders to move them into as a way for me to more easily find the type of notes I'm after.
The counterpoint to this is the search bar that can already be used to find my notes for the specific room or genre I want to some degree.
